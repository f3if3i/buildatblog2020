How to run

clone the repository
```
$ git clone https://github.com/lottie0914-faye/buildatblog2020.git
$ cd buildatblog2020
```
install packages and create virtual environment:
```
$ pip3 install pipenv
$ pipenv install 
$ pipenv shell
```
initialize admin and run flask
```
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```
